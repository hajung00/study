    # 순서도 그리기
        1. 테이블 생성 후 칸에 랜덤하게 지뢰 심기
        2. 왼쪽 마우스 클릭 할 경우
            2-1. 물음표나 깃발이 있는 경우 클릭 할 수 없게 대기
            2-2. 폭탄일경우, 게임 종료
            2-3. 폭탄이 아닐 경우, 칸을 열고 주변 칸의 개수를 표시
            2-4. 주변 칸 수가 0일 경우, 열 수 있으면 열고 아닐경우 대기
            2-5. 2-4에서 열었을경우 모든 칸을 다 열었는지 체크
        3. 오른쪽 마우스 클릭 할 경우
            3-1. 이미 열린칸이 아닐경우
            물음표일 경우 => 깃발로 전환
            깃발일 경우 => 닫힌칸으로 
            깃발이 아닐 경우 => 물음표로 전환
     
* 논리 연산자
 - && : 앞에가 true면 뒤에 실행
 - || : 앞에가 false면 뒤에 실행
 - ?? : 앞에가 null또는 undifind면 뒤에 실행
