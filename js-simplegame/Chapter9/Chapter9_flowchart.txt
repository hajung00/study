1. tictactoe 순서도 그리기
    # 순서도 그리기
        1. 3x3이차원 배열 준비
        2. 누구의 턴으로 할지 설정
        3. 화면에 그려줌
        4. 칸을 클릭하면 해당 칸에 표시
        5. 승부가 낫을 경우, 승자를 표시
        6. 승부가 나지 않았을 경우, 다음 턴으로 넘겨줌(승부가 날때까지 반복)
     
2. tictactoe self 순서도 그리기
    # 순서도 그리기
        1. 3x3이차원 배열 준비
        2. 누구의 턴으로 할지 설정
        3. 화면에 그려줌
        4. 칸을 클릭하면 해당 칸에 표시
        5. 승부가 낫을 경우, 승자를 표시
        6. 승부가 나지 않았을 경우, 다음 턴으로 넘겨줌
        7. 컴퓨터의 턴이면 한칸을 그린다.
        8. 컴퓨터의 턴이 아닐경우 4로 가서 반복한다.
