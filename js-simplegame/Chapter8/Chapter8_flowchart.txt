1. 반응속도 체크 순서도 그리기
    # 순서도 그리기
        1. 파란 화면 띄우고 대기
        2. 파란 화면을 클릭하면 빨간 화면으로 전환 후 랜덤 타이머 작동
        3. 자동으로 초록화면이 뜨기 전에 빨간 화면을 클릭하면 성급했다는 메세지 출력
        4. 초록 화면 클릭시 클릭까지의 시간 측정 후 표시
        5. 파란 화면으로 전환

2. 반응속도 체크 순서도 그리기(속도 순으로 정렬)
    # 순서도 그리기
       1의 5까지 동일
       + 속도를 저장해 놓은 변수를 오름 차순으로 정렬
       + 인덱스 0부터 4까지 자름
       + 화면에 결과 표시

* classList
- contains: 해당 클래스의 존재 확인 
- add: 해당 클래스 추가
- replace: 해당 클래스 수정
- remove: 해당 클래스 제거
ex) 태그.classList.contains('클래스')
