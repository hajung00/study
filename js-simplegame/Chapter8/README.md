## Chapter8

### 1. 반응 속도 구하기

#### - 게임 설명

- 1. 처음의 파랑 화면은 대기 화면을 의미
  2. 화면을 한번 클릭하면 빨강 화면으로 전환되며 준비 화면을 의미
  3. 준비 화면에서는 임의의 시간이 지난 후에 초록 화면으로 자동 전환
  4. 초록 화면을 보자마자 클릭하면 초록 화면이 뜬 시각과 클릭한 시각의 차이를 구해 반응속도 측정
  5. 만약 준비 화면일 때 클릭하면 성급했다는 메세지 출력 후 대기 화면으로 간다.
